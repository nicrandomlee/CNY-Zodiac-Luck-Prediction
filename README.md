# 2026 Chinese Zodiac Luck Prediction

A premium, interactive web application to check your luck for the year 2026 based on your Chinese Zodiac. This application generates personalized advice for Wealth, Career, Relationships, and Health, along with your lucky number and color.

## Features
- **Interactive Zodiac Grid**: Select your animal from a beautifully animated grid.
- **Detailed Predictions**: Get randomized insights for 2026 tailored to specific life categories.
- **Premium Design**: Built with a modern, responsive UI using Tailwind CSS.

## Installation & Usage

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run the Application**
   ```bash
   npm run dev
   ```
   The application will start at `http://localhost:8501` (or 8502 if 8501 is busy).

3. **Build for Production**
   ```bash
   npm run build
   ```
